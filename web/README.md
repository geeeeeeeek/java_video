#### 部署步骤

1. 修改constants.ts中的BASE_URL
2. vite build
3. 将dist部署到nginx


#### 配置解释

1. env.development 开发环境配置
2. eslintrc.js 代码规范化提示
3. vite.config.js vite 开发服务器配置
